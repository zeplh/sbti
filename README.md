# SBTI —— 什么是 SBTI 人格测试？

SBTI 全称 **Silly Big Personality Test**（傻大个人格测试），是 2026 年初在中文互联网上迅速走红的一款**娱乐性人格测试**。它模仿 MBTI 的框架，但完全不追求心理学严谨性，取而代之的是大量网络热梗、自嘲文化和抽象表达。

## 起源

SBTI 最初由 B 站 UP 主 **@蛆肉儿串儿** 发布，是一个纯前端网页测试。用户回答若干道选择题后，系统会给出一个“人格类型”——例如 **拿捏者、伪人、傻乐者、酒鬼、吗喽** 等。

2026 年 4 月，SBTI 在微信朋友圈形成刷屏级传播，成为当年第一个单一素材的朋友圈刷屏案例。

## 核心机制

- **15 个评估维度**：分为自我、情感、态度、行动、社交五大模型，每个模型下 3 个维度。
- **曼哈顿距离匹配**：将用户的 15 维得分（低/中/高）转化为向量，与 25+ 种人格模板比对，找出最接近的类型。
- **隐藏彩蛋**：存在极少数触发条件的隐藏人格（如“酒鬼 DRUNK”）。

## 与 MBTI 的区别

| 维度 | MBTI | SBTI |
|------|------|------|
| 目的 | 职业/性格参考 | 娱乐、社交、玩梗 |
| 科学性 | 有心理学基础（虽存争议） | 无，纯娱乐 |
| 结果风格 | 严肃、中性 | 戏谑、自嘲、抽象 |
| 传播方式 | 专业测评/企业/教育 | 朋友圈、社交媒体截图 |

## 为什么火了？

- **低门槛**：纯前端，点开即测。
- **强共鸣**：人格标签符合年轻人自嘲心理（“酒鬼”“吗喽”等）。
- **可传播**：结果页设计适合截图发朋友圈，自带分享动力。
- **模因属性**：每个类型都像是一个社交模因，容易引发讨论和二次创作。

## 注意事项

- SBTI 仅供娱乐，**不可用于**心理诊断、教育评估、招聘决策等严肃场景。
